## 📌 核心问题

给定目标分布 $p(x)$，如何高效生成服从该分布的独立样本 $\{x_1, x_2, \dots, x_N\}$？

---

## 1️⃣ 逆变换采样 (Inverse Transform Sampling)

### 原理

利用累积分布函数 (CDF) 的逆变换：

$$
F(x) = \int_{-\infty}^{x} p(t) dt
$$

若 $U \sim \text{Uniform}(0,1)$，则 $X = F^{-1}(U)$ 服从分布 $p(x)$。

### 算法

```python
def inverse_transform_sampling(p, F_inv, n_samples):
    samples = []
    for _ in range(n_samples):
        u = random.uniform(0, 1)
        x = F_inv(u)  # 需要 CDF 的反函数
        samples.append(x)
    return samples
```

### 优缺点

| 优点 | 缺点 |
|------|------|
| 简单直观 | 需要 $F^{-1}$ 的解析表达式 |
| 样本独立 | 很多分布没有闭式逆 CDF |

### 示例：指数分布

$$
p(x) = \lambda e^{-\lambda x}, \quad F(x) = 1 - e^{-\lambda x}
$$

$$
F^{-1}(u) = -\frac{1}{\lambda} \ln(1-u)
$$

## 2️⃣ 拒绝采样 (Rejection Sampling)

### 原理

用易于采样的提议分布 $q(x)$ 包裹目标分布 $p(x)$，通过接受/拒绝来筛选样本。

需要找到常数 $M$ 使得：$M \cdot q(x) \geq p(x), \forall x$

### 算法

```python
def rejection_sampling(p, q, q_sample, M, n_samples):
    samples = []
    while len(samples) < n_samples:
        x = q_sample()  # 从 q(x) 采样
        u = random.uniform(0, 1)
        if u < p(x) / (M * q(x)):
            samples.append(x)  # 接受
        # 否则拒绝，继续循环
    return samples
```

### 接受概率

$$
P(\text{accept}) = \frac{1}{M}
$$

$M$ 越小效率越高，理想情况 $M \approx 1$。

### 优缺点

| 优点 | 缺点 |
|------|------|
| 不需要 $F^{-1}$ | 高维时效率极低 |
| 样本独立 | 需要选择合适的 $q(x)$ 和 $M$ |

## 3️⃣ 蓄水池采样 (Reservoir Sampling)

### 问题

从**数据流**或**未知大小**的总体中均匀采样 $k$ 个样本。

### 算法 (Algorithm R)

```python
def reservoir_sampling(stream, k):
    reservoir = []
    for i, item in enumerate(stream):
        if i < k:
            reservoir.append(item)  # 前 k 个直接放入
        else:
            j = random.randint(0, i)
            if j < k:
                reservoir[j] = item  # 以 k/(i+1) 概率替换
    return reservoir
```

### 正确性证明

第 $i$ 个元素最终被保留的概率：

$$
P(\text{保留}) = \frac{k}{i+1} \times \prod_{j=i+1}^{n} \left(1 - \frac{k}{j+1} \times \frac{1}{k}\right) = \frac{k}{n+1}
$$

### 应用

- 在线学习
- 大数据流处理
- 日志采样

## 4️⃣ 马尔可夫链蒙特卡洛 (MCMC)

### 核心思想

构造一个马尔可夫链，其平稳分布为目标分布 $p(x)$。

### 4.1 Metropolis-Hastings 算法

```python
def metropolis_hastings(p, q_proposal, x0, n_samples, burn_in=1000):
    samples = []
    x = x0
    for i in range(n_samples + burn_in):
        x_star = q_proposal(x)  # 提议新状态
        alpha = min(1, p(x_star) * q_proposal(x, x_star) / 
                       (p(x) * q_proposal(x_star, x)))
        if random.uniform(0, 1) < alpha:
            x = x_star  # 接受
        if i >= burn_in:
            samples.append(x)
    return samples
```

### 接受率

$$
\alpha(x, x^*) = \min\left(1, \frac{p(x^*) q(x^* \to x)}{p(x) q(x \to x^*)}\right)
$$

### 4.2 Gibbs 采样

M-H 的特例，每次只更新一个维度，接受率恒为 1。

$$
x_i^{(t+1)} \sim p(x_i | x_1^{(t+1)}, \dots, x_{i-1}^{(t+1)}, x_{i+1}^{(t)}, \dots, x_d^{(t)})
$$

### 优缺点

| 优点 | 缺点 |
|------|------|
| 适用于高维复杂分布 | 样本相关（非独立） |
| 不需要归一化常数 | 需要 burn-in 和 thinning |

## 5️⃣ 重要性采样 (Importance Sampling)

### 问题

估计期望：$\mathbb{E}_{p}[f(x)] = \int f(x) p(x) dx$

### 原理

用提议分布 $q(x)$ 重写积分：

$$
\mathbb{E}_{p}[f(x)] = \int f(x) \frac{p(x)}{q(x)} q(x) dx = \mathbb{E}_{q}\left[f(x) w(x)\right]
$$

其中权重：$w(x) = \frac{p(x)}{q(x)}$

### 估计量

$$
\hat{\mu} = \frac{1}{N} \sum_{i=1}^{N} f(x_i) w(x_i), \quad x_i \sim q(x)
$$

### 自归一化重要性采样

当 $p(x)$ 未归一化时（只知道 $\tilde{p}(x) = Z \cdot p(x)$）：

$$
\hat{\mu} = \frac{\sum_{i=1}^{N} f(x_i) w(x_i)}{\sum_{i=1}^{N} w(x_i)}, \quad w(x_i) = \frac{\tilde{p}(x_i)}{q(x_i)}
$$

### 关键

$q(x)$ 应尽量接近 $p(x)$，尤其在 $f(x)p(x)$ 大的区域。

## 6️⃣ 重要性重采样 (Importance Resampling)

### 思想

先做重要性采样，然后根据权重**重采样**得到近似独立样本。

### 算法

```python
def importance_resampling(p, q, q_sample, n_samples):
    # 第一步：重要性采样
    particles = []
    weights = []
    for _ in range(n_samples):
        x = q_sample()
        w = p(x) / q(x)
        particles.append(x)
        weights.append(w)
    
    # 归一化权重
    w_sum = sum(weights)
    weights = [w / w_sum for w in weights]
    
    # 第二步：根据权重重采样
    resampled = []
    for _ in range(n_samples):
        idx = random.choices(range(n_samples), weights=weights)[0]
        resampled.append(particles[idx])
    
    return resampled
```

### 应用

- 粒子滤波 (Particle Filter)
- 序贯蒙特卡洛 (SMC)
- 贝叶斯推断

## 📊 方法对比

| 方法 | 样本独立 | 高维适用 | 需要归一化 | 效率 |
|------|----------|----------|------------|------|
| 逆变换 | ✅ | ❌ | ✅ | 高（若有 $F^{-1}$） |
| 拒绝采样 | ✅ | ❌ | ✅ | 中（依赖 $M$） |
| 蓄水池 | ✅ | ✅ | ✅ | 高 |
| MCMC | ❌ | ✅ | ❌ | 中（需 burn-in） |
| 重要性采样 | ✅ | ⚠️ | ❌ | 依赖 $q$ 的选择 |
| 重要性重采样 | ⚠️ | ⚠️ | ❌ | 中 |

## 💻 图形学应用

### 路径追踪中的重要性采样

- **BRDF 采样**：按反射分布采样光线方向
- **光源采样**：优先采样贡献大的光源
- **多重重要性采样 (MIS)**：结合多种采样策略

```glsl
// 余弦加权半球采样（Lambertian 表面）
vec3 cosine_weighted_hemisphere(vec3 normal) {
    float r1 = random();
    float r2 = random();
    float phi = 2.0 * PI * r1;
    float cosTheta = sqrt(r2);
    float sinTheta = sqrt(1.0 - cosTheta * cosTheta);
    
    // 转换到世界空间...
    return vec3(sinTheta * cos(phi), sinTheta * sin(phi), cosTheta);
}
```
